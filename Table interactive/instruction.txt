Инструкция по приминению:


//----------Отображение данных
1. Пользователь нажимает кнопку SMALL_DATA (или BIG_DATA в зависимости от объема запрашиваемых данных)
2. Система обрабатывает запрос и через некоторое время отображает полученнные данные в таблице. Активируется кнопка ADD_RECORD


//----------Добавление новой записи в таблицу
1. Пользователь нажимает на кнопку ADD_RECORD для добавления новой записи в таблицу
2. Система отображает форму для добавления новой записи
3. Пользователь заполняет форму для добавления новой записи 
4. Система активирует кнопку ADD после заполнения всех полей
5. Пользователь нажтмает кнопку ADD
6. Система валидирует введенные пользователем значения. Если присутствуют ошибки, система оповещает пользователя через диалоговое окно. Если ошибок в заполнении полей нет, то система добавляет запись в таблицу.


//----------Дополнительная информация о записи
1. Пользователь нажимает на запись в таблице
2. Система отображает окно с таблицей, в которой приведена дополнительная информация о выбранной записи
3. Пользователь кликает мышью вне окна с дополнительной информацией о записи
4. Система скрывает окно с дополнительной информацией о записи


//----------Поиск по таблице
1. Пользователь вводит значения в поисковое поле и нажимает кнопку SEARCH.
2. Система ищет по всем полям таблицы введенное пользователем значение. Если совпадения есть, то сиситема выводит на экран результаты поиска. Если совпадений нет, то система отображает пустую таблицу.
3. Система отображает кнопку BACK для возврата к полной таблице
4. Пользователь нажимат кнопку BACK и возвращается на основную страницу с полной таблицей.


//----------Сортировка по Id
1.1 Пользователь нажимает Id в заголовке таблице.
2.1 Система сортирует записи по возрастанию Id.

1.2 Пользователь нажимает 2 раза Id в заголовке таблице.
2.1 Система сортирует записи по убыванию Id.

//----------Сортировка по Id
1.1 Пользователь нажимает FirstName в заголовке таблице.
2.1 Система сортирует записи в алфавитном порядке по значению FirstName.

1.2 Пользователь нажимает FirstName в заголовке таблице.
2.2 Система сортирует записи в обратном алфавитном порядке по значению FirstName.
